# Utilify

A modern browser-based utility suite containing practical tools for developers, designers, writers, students, and professionals.

Every tool performs its processing locally on your device — no accounts, no backend, no data collection.

## Tools

### Text
- **Text Analyzer** — Character, word, sentence, paragraph counts + reading time
- **Case Converter** — UPPERCASE, lowercase, Title Case, camelCase, snake_case, kebab-case
- **Markdown Preview** — Live rendering with HTML export
- **Lorem Ipsum Generator** — Generate placeholder text by paragraphs, sentences, or words
- **Slug Generator** — URL-safe slug generation

### Developer
- **JSON Formatter** — Format, minify, and validate JSON
- **JWT Decoder** — Decode and inspect JSON Web Tokens
- **Base64 Tool** — Encode and decode Base64
- **URL Encoder** — Encode and decode URLs
- **Regex Tester** — Live regex matching with highlighting
- **Hash Generator** — SHA-256, SHA-384, SHA-512, SHA-1 via Web Crypto API
- **UUID Generator** — Single or bulk UUID generation
- **Timestamp Converter** — Unix ↔ human-readable date conversion

### Design
- **Color Palette Generator** — Random palettes with lock and copy
- **Color Converter** — HEX ↔ RGB ↔ HSL
- **Contrast Checker** — WCAG AA and AAA compliance
- **Gradient Generator** — CSS gradient editor with export
- **Box Shadow Generator** — CSS box-shadow editor with live preview
- **Border Radius Generator** — Independent corner controls with CSS export

### Utilities
- **Unit Converter** — Length, weight, temperature, area, volume
- **Timezone Converter** — Multi-timezone comparison
- **Password Generator** — Custom length, symbols, strength indicator
- **QR Generator** — QR codes with PNG/SVG export
- **QR Scanner** — Scan QR codes from uploaded images

## Features

- **Command Palette** — `Ctrl+K` to navigate to any tool instantly
- **Search** — `/` to open search, instant filtering on homepage
- **Dark Mode** — System preference detection + manual toggle
- **Favorites** — Pin tools with localStorage persistence
- **Recently Used** — Auto-tracked tool history
- **View Transitions** — Smooth page navigation
- **Scroll Reveals** — Intersection Observer animations

## Tech Stack

- HTML5
- CSS3 (custom properties, no framework)
- Vanilla JavaScript (ES6+)
- Web Crypto API
- View Transitions API
- Intersection Observer
- LocalStorage

## Hosting

Static deployment — compatible with GitHub Pages.
